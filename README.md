# Ultra Camera Pro

A professional-grade web camera application built with pure HTML, CSS, and JavaScript.

## Features
- Photo & Video Recording
- Pro Mode (Manual ISO, EV, WB)
- Portrait Mode Simulation
- Zoom Controls (0.5x to 100x)
- Tap to Focus & Pinch to Zoom
- Grid Overlay
- PWA Support

## How to Run
1. Serve the directory using a local web server (e.g., `npx serve .` or Vite).
2. Open in a modern browser.
3. Grant camera and microphone permissions.

## GitHub Pages Deployment
1. Push the code to a GitHub repository.
2. Go to Settings > Pages.
3. Select the `main` branch and `/` root folder.
4. Save and wait for deployment.
